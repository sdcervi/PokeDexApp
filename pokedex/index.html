I’ve made a few changes to how I’m managing feature requests. They used to be through GitHub Issues, but some of you were having trouble accessing that, and the Google Forms feature request experience is clunky at best.

I’m happy to announce that feature requests are now managed through Changemap! This should make it a lot easier for everyone to make new feature requests, comment on and upvote existing feature requests, and subscribe to specific tickets for updates on them.

Access the new Changemap here.

If you want to get updates when I send them out rather than having to check this banner, you can either subscribe to the newsletter (free or paid) or become a paid supporting member.
