How do I use this?

Toggle between List and Grid view

You can either use the default grid view, where Pokémon are arranged in boxes just like in Pokémon Home, or the list view where each Pokémon and all its variants are on the same row of a list. Use the buttons in the upper left to switch between views.

Alt forms checklist

Each dex type (normal or shiny) also has an alt/variants section. For Alcremie’s alt forms (and others, coming soon), you can use the Alt Forms Guide page to help you track which alt forms you need and how to get the remaining forms. Please note that this does not retain progress like the PokéDex page does.
